# UNIT-PROJECT-1

## Project : Personal Banking Manager :

#### Overview : Personal Banking Manager is a CLI-based application that allows users to manage all their personal bank accounts in one place and users can track balances, incomes, expenses, and debts, as well as analyze their financial situation through summaries and reports.

#### Main User : Account Owner – the person who manages their accounts and financial data.

### Features & User Stories
#### As a Account Owner I should be able to do the following :
- Register , login using my national ID and logout. 
- Add, view,update and delete multiple bank accounts.
- Record income and expenses per account.
- Add and pay debts.
- View total balances, total expenses, and total debts per account. 
- Calculate Net Worth. 
- Display a financial summary table with bank-wise balances, expenses, debts, and percentage of total wealth. 
- View a Financial Health Score.
- Generate PDF report 

#### Usage :
 - type in register_user to register a new user
 - type in login_user to login as a user
 - type in logout_user to logout of current session.
 - type in add_account to add a new bank account
 - type in show_accounts to show all bank accounts
 - type in delete_account to delete a bank account.
 - type in update_account to update a bank account.
 - type in add_income to add income to a bank account
 - type in add_expense to add an expense to a bank account.
 - type in add_debt to add a debt to a bank account 
 - type in pay_debt to pay a debt partially or fully
 - type in financial_summary to display a table summary of all accounts
 - type in net_worth to calculate total net worth
 - type in financial_score to display Financial Health Score
 - type in generate_report to generate PDF report.
 - type in exit to exit the system.
